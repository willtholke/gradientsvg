# gradientsvg

A web app for generating and exporting beautiful, infinitely scalable gradient SVGs.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Built with

- Next.js 15 + React 18
- Tailwind CSS
- Shadcn UI components
- Lucide icons
- Framer Motion
- React Parallax Tilt
